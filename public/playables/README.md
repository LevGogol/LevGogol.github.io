# Playable Ads

Здесь будут размещены интерактивные (playable) рекламные мини-игры / демо.
Каждая папка = отдельный билд (HTML/CSS/JS), доступный по статическому URL после деплоя.

Структура (пример):

```
playables/
  my-first-playable/
    index.html
    assets/
      ...
  puzzle-mini/
    index.html
```

Рекомендации:

- Один `index.html` для быстрого старта.
- Минимизируйте размер: < 2MB желательно.
- Отложите загрузку тяжёлых ассетов.
- Добавьте `console.log` с версией билда.

После добавления папки вы сможете ссылаться на неё как `/playables/<folder>/`.
